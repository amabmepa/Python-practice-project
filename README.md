# پروژه تحلیل وضعیت آب‌وهوا (خروجی فارسی)

این پروژه یک برنامه ساده تحلیل آب‌وهوا است که با زبان Python پیاده‌سازی شده و در محیط Google Colab اجرا می‌شود.

## شرح پروژه
کاربر نام یک شهر را وارد می‌کند و برنامه با استفاده از OpenWeatherMap API اطلاعات دما، رطوبت و وضعیت هوا را دریافت می‌کند. داده‌ها پردازش شده و خروجی شامل نمایش متنی و نمودار است.

## ساختار پروژه
- `api/` : کد ارتباط با API
- `visualization/` : رسم نمودار
- `notebooks/` : اجرای پروژه در Google Colab
- `data/` : نمونه داده‌ها
- `tests/` : تست کدها
- `requirements.txt` : کتابخانه‌های مورد نیاز

## ابزارها
- Python
- Google Colab
- requests
- matplotlib

## نحوه اجرا
1. فایل پروژه را در Google Colab باز کنید
2. کدها را اجرا کنید
3. نام شهر به انگلیسی وارد کنید
4. خروجی متنی و نمودار نمایش داده می‌شود

## خروجی برنامه
- دمای شهر
- رطوبت
- وضعیت هوا به زبان فارسی
- نمودار دما
